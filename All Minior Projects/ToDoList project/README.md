# Staring with the Project
### General steps to follow when creating a project

- Step1: Developing the UI/Design.
- Step2: Functionality

	- Delete a TODO
	- check task

- Step3: Data
	- tasks - an array
	- task - {done, text, id}
- Step4: Functions (in code) ie in javaScript
	- addTodo
	- deleteTodo
	- checkTodo
	- renderTodosList
	- showNotification
